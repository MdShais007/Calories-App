🍽️ Calories Calculator (Prototype)

A simple Django-based web application that calculates calorie intake based on food and quantity.
⚠️ This is a **demo/prototype project** built for learning purposes.

---

 Features

 🔍 Search food by name
 
 ⚖️ Enter quantity (in grams)
 
 📊 Calculate total calories
 
 🗂️ Uses a small sample dataset (5 food items)
 
 💻 Basic UI for demonstration


  Project Status

This project is a **prototype**:

* Only **5 food items** are stored in the database
* No external API or large dataset used
* Focus is on **Django basics (models, views, forms)**


 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS
* **Database:** SQLite
* **Version Control:** Git & GitHub



 Usage

1. Enter food name (e.g., rice)
2. Enter quantity in grams
3. View calculated calories


 📈 Future Improvements

* Integrate real food API (e.g., nutrition database)
* Add more food items dynamically
* User login & daily tracking
* Better UI/UX
* Deploy the app
